Лабораторная 1

Задание: Необходимо создать сайт отчет и залить его на хостинг GitHub Pages
Действия по создание и деплою на хостинг
  1. Был создан новый репозиторий на сайте github pages
  2. Была создана новая папка, в которой хранятся рабчие файла сайта
  3. Команды для деплоя сайта на хостинг
    • git init - создает локальный репозиторий в нашей папке
    • git add . - добавление всех новых файлов в отслеживание
    • git commit -m"commit text" - создания комментария к нашим изменениям
    • git remote add origin ссылка_на_репозиторий - связываем наш локальный репозиторий с удаленным
    • git push -u origin master - пушим наши зменения на удаленный репозиторий
    • git checkout -b gh-pages - создаем новую ветку gh-pages, которая является хостингом
    • git checkout master - возвращаемся на ветку master
    • git push origin master:gh-pages - пушим все файлы из ветки msater в ветку gh-pages, тем самым заливая файлы на хостинг

Лабораторная 3

Занимался разработкой Frontend части сайта AppleFix.

Сначала было необходимо сверстать разметку главной страницы сайта, которая представляет из себя Лендинг.

HTML - разметка

Для этого используется HTML

HTML (HyperText Markup Language) – язык гипертекстовой разметки документов для создания структуры страницы: заголовки, абзацы, списки, именуемые «тегами».

При разметки страницы были соблюдины правила семантики.

Семантический элемент четко описывает свое значение как для браузеров, так и для разработчиков

CSS - таблицы стилей

Чтобы размтека соответствовала макету, который составил жизайнер используется CSS.

CSS (Cascading Style Sheets) – формальный язык описания внешнего вида документа, написанного с использованием языка HTML. Благодаря CSS-коду браузер понимает, как именно отображать элементы. CSS задаёт цвета и параметры шрифтов, определяет, как будут располагаться разные блоки сайта, и так далее.

Для упрощения стилизации и переиспользования блоков использовалась методология БЭМ, разработанная в Яндекс.

БЭМ (Блок, Элемент, Модификатор) — компонентный подход к веб-разработке. В его основе лежит принцип разделения интерфейса на независимые блоки. Он позволяет легко и быстро разрабатывать интерфейсы любой сложности и повторно использовать существующий код, избегая «Copy-Paste».

Чтобы сайт компании хорошо отображался на всех устройствах, дизайнером был создан макет под десктоп, таблет и мобайл.
Медиа-запросы
Для верстки этих макетов были использованы медиа-запросы.

Возможности медиа-запросов не ограничиваются выявлением мобильных устройств, с их помощью можно создавать адаптивный макет. Такой макет подстраивается под разрешение монитора и окна браузера, меняя при необходимости ширину макета, число колонок, размеры изображений и текста. Медиа-запросы ограничивают ширину макета и при достижении этого значения (к примеру за счёт уменьшения окна или при просмотре на устройстве с указанным размером) уже применяется другой стиль.

JavaScript

Для того, чтобы оживить сайт используется JavaScript

JavaScript – это язык программирования, который позволяет реализовать сложное поведение веб-страницы. Он используется как при front-end разработке, при которой главная задача – динамически отображать контент, реагировать на действия пользователя, обрабатывать клики мышкой, перемещения курсора, нажатия клавиш и т.д., так и при back-end разработке, чтобы посылать запросы на сервер и взаимодействовать с базой данных. В решениях проекта, JavaScript применяется для разработки калькулятора расчета стоимости ремонта, услуги Trade-In, добавлении корзины покупателя страницы магазина, и других функциональных модулей сайта. Данные разработки позволят новому интернет-ресурсу компании ускорить процессы оформления услуги.
